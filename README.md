# iPlant — AI Assistant (Static Site)

A simple static web UI for the iPlant AI Assistant.

## Quick Start
- Open `index.html` in any modern web browser.
- No build tools or backend required.

## Files
- `index.html` — main page and layout
- `assets/css/styles.css` — styles
- `assets/js/app.js` — basic interactivity
- `logo.png`, `logo5.png` — branding icons/favicons

## Notes
- Update the logo images if you change branding.
- Ensure all files remain in the same folder structure.
